## Featured Project

### BiliSum

> AI video summarizer and knowledge base for Bilibili, YouTube and local videos.

BiliSum 是我目前重点打磨的 AI 产品项目。  
它可以把 B 站、YouTube 和本地视频转化为结构化知识笔记，支持转写、摘要、知识卡片、思维导图、RAG 问答和 Markdown / Obsidian 导出。

核心能力：

- 视频探测、下载、转写与摘要生成
- 支持 Bilibili / YouTube / 本地视频
- SiliconFlow ASR / 本地 Whisper 可选
- OpenAI-compatible LLM 接口
- 跨视频知识库检索与问答
- Electron + React 桌面端
- FastAPI + SQLite 本地服务
- Docker / electron-builder / PyInstaller 打包分发

技术栈：

```text
Frontend    Electron · React · TypeScript · Vite
Backend     Python · FastAPI · SQLite
AI          Whisper · SiliconFlow ASR · LLM API · RAG
Infra       Docker · PyInstaller · electron-builder
```
